# 版本说明

## 当前版本 V0.2（未发布）

1、新增“灵感罐子

”开启「灵感罐子」：随机选择模式

- 不用想关键词
- 不用打字
- 点一下"推荐"按钮
- 交给命运就好！

2、新增近义词典

原始输入："今天发工资了"

第一步：近义词扩展：包含"工资"

触发近义词映射 → 扩展出 ["开心", "高兴", "快乐", "奖励"]

第二步：逐个匹配计分：汉堡规则标签：["开心", "奖励"]
"开心" 是近义词命中 → +1分

"奖励" 是近义词命中 → +1分

本项总分 = 2分 ✨

 其他所有规则 = 0分

✅ 结果：推荐「牛肉汉堡 + 薯条」

| 匹配方式 | 分值 | 说明 |
| :--- | :--- | :--- |
| 直接命中标签 | +2 分 | 用户输入直接包含了某个规则的标签 |
| 近义词命中 | +1 分 | 通过近义词间接匹配上标签 |
| 长标签权重 | ×1.2 | 3字以上的标签，权重额外提升 |

每次重新输入分数会清0

3、增加食物名称匹配功能

用户输入：肉，名称或标签带有“肉”字的食物的加分会更高

4、新增了标签工坊的重置和清空功能
